# AWS EC2 Web Server Deployment

**Description:**  
Deployed a live website on AWS EC2 using Apache HTTP Server. This project demonstrates cloud server setup, Linux server management, and website hosting.

## Steps Completed
1. Launched an EC2 t3.micro instance on AWS.
2. Connected securely using SSH.
3. Installed Apache HTTP Server and started the service.
4. Configured security groups to allow HTTP (port 80) traffic.
5. Deployed a static HTML website.
6. Verified website is live using EC2 public IP.

## Skills Demonstrated
- AWS EC2
- Linux / SSH
- Apache HTTP Server
- Web server configuration
- Networking basics
